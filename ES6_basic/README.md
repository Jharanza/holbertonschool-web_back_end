<h1>ES6 Basics</h1>
<p>In this proyecto, we learn about</p>
<ul>
    <li>What ES6 is</li>
    <li>New features introduced in ES6</li>
    <li>The difference between a constant and a variabl</li>
    <li>Block-scoped variables</li>

    <li>String templating in ES6</li>
